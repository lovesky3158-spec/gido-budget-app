# Dashboard / Report filter patch

## 반영 내용

1. Dashboard 상단 필터에 카테고리 필터 추가
   - PC: 사용자 / 카테고리 / 결제수단 순서로 표시
   - 모바일: 필터 모달 안에 카테고리 섹션 추가
   - 초기화 시 사용자, 결제수단, 카테고리 필터 모두 초기화

2. 카테고리별 소비분석 카드 클릭 시 카테고리 필터 적용
   - 기존 activeCategory 선택과 함께 categoryFilter도 같이 변경
   - 전체 버튼 클릭 시 카테고리 필터 해제

3. 결제수단별 지출현황 카드 클릭 시 결제수단 필터 적용
   - 세부 결제수단 카드 클릭 시 해당 결제수단으로 리포트 전체 재계산
   - 전체 카드 클릭 시 결제수단 필터 해제
   - 선택된 결제수단은 강조 표시
